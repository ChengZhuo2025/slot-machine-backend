## 1. 阿里云账户主体变更

### 📝 工作说明

您现有的阿里云账户绑定的是另一家企业的主体信息，为了后续系统正常使用和财务结算，需要将阿里云账户的主体变更为娃娃项目实际经营的公司。

### ✅ 需要做什么

1. **准备变更材料**
   - [ ] 贵公司营业执照（扫描件或拍照）
   - [ ] 法人身份证正反面照片
   - [ ] 企业对公银行账户信息

2. **提交变更申请**
   - [ ] 登录阿里云官网（www.aliyun.com）
   - [ ] 进入"账号管理 → 实名认证"
   - [ ] 点击"企业实名认证变更"
   - [ ] 按照提示上传相关材料
   - [ ] 等待审核（通常3-5个工作日）

### 💰 费用说明

主体变更本身不收费

### ⏰ 预计完成时间

提交后3-5个工作日（含审核时间）

## 2. 域名注册

### 📝 工作说明

系统需要使用域名来提供服务，包括：
- 后端 API 接口服务
- 管理后台（运营管理系统）
- 用户端小程序接口

建议注册一个主域名，然后使用二级域名来区分不同的服务。

### ✅ 需要做什么

#### 第一步：准备域名

- [ ] 确定要注册的域名（建议与品牌相关）
  - 例如：`dumeirei.com`（爱上杜美人拼音）
  - 或者：`asdmr.com`（爱上杜美人首字母）

💡 **域名选择建议：**
- 简短易记，便于推广
- 与品牌名称相关
- 优先选择 `.com` 域名
- 避免使用中横线、数字等不利于记忆的字符

#### 第二步：在阿里云购买域名

- [ ] 登录阿里云控制台
- [ ] 搜索"域名注册"
- [ ] 输入您想要的域名，查询是否可注册
- [ ] 如果可注册，点击"加入清单"并完成购买
- [ ] 填写域名持有者信息（建议填写企业信息）
- [ ] 完成支付

#### 第三步：域名实名认证

- [ ] 购买完成后，进入"域名控制台"
- [ ] 找到刚购买的域名，点击"实名认证"
- [ ] 选择认证类型：**企业认证**
- [ ] 上传企业营业执照
- [ ] 填写企业信息
- [ ] 提交审核（通常1个工作日内完成）

⚠️ **重要提示：** 域名必须完成实名认证后才能正常使用，否则会被锁定

#### 第四步：域名备案（如使用国内服务器）

如果服务器部署在国内（阿里云、腾讯云等），域名必须完成ICP备案：

- [ ] 登录阿里云备案系统
- [ ] 选择"首次备案"
- [ ] 按照提示填写备案信息
- [ ] 上传相关资质材料
- [ ] 等待管局审核（通常10-20个工作日）

💡 **备案小知识：**
- 备案期间域名不能访问
- 建议提前进行备案，避免影响上线
- 如使用香港或海外服务器，则不需要备案

#### 第五步：提供域名信息给我

域名购买完成后，请告知我：

- [ ] 域名名称：`____________`（如：dumeirei.com）
- [ ] 域名注册商：阿里云（万网）
- [ ] 域名状态：☐ 实名认证完成 ☐ 备案完成（国内服务器）

💡 **我会做什么：** 域名解析配置（将域名指向服务器）由我来完成，您无需操作

### 💰 费用说明

- **域名注册费：**
  - `.com` 域名：约 55-69 元/年（首年）
  - `.cn` 域名：约 29-38 元/年（首年）
  - 续费价格通常略高于首年价格

- **域名备案费：** 免费（如需要）

- **SSL 证书：** 免费（使用 Let's Encrypt 免费证书）

### ⏰ 预计完成时间

- 域名注册：10分钟
- 实名认证：1个工作日
- ICP备案：10-20个工作日（如需要）

### ⚠️ 注意事项

1. 域名注册后不能退款，请确认好再购买
2. 域名每年需要续费，建议开启自动续费，避免过期
3. 域名持有者信息建议填写企业信息，便于后续管理
4. 如果域名已被注册，可以考虑其他后缀（如 .cn、.net）或修改域名

### 📋 推荐的域名规划

购买主域名后，我们会配置以下二级域名：

| 二级域名 | 用途 | 示例 |
|---------|------|------|
| api.您的域名.com | 后端API服务 | api.dumeirei.com |
| admin.您的域名.com | 管理后台 | admin.dumeirei.com |
| cdn.您的域名.com | CDN静态资源 | cdn.dumeirei.com |

- 酒店图片

10 分钟内即可开通

## 5. 注册和认证微信小程序

### 📝 工作说明

微信小程序是用户使用系统的主要入口，需要注册一个企业主体的小程序账号。

### ✅ 需要做什么

#### 第一步：注册小程序账号

- [ ] 访问微信公众平台（https://mp.weixin.qq.com/）
- [ ] 点击"立即注册"
- [ ] 选择"小程序"类型
- [ ] 填写企业邮箱（注意：不能使用QQ、163等个人邮箱）
- [ ] 填写管理员信息（需要扫码验证）
- [ ] 选择主体类型：**企业**
- [ ] 填写企业信息并上传营业执照

#### 第二步：企业认证（重要）

- [ ] 登录小程序后台
- [ ] 在"设置"中找到"微信认证"
- [ ] 点击"开通"并按流程完成认证
- [ ] 支付认证费用：**300元/年**（腾讯收取，必须缴纳）

⚠️ **为什么必须认证？**
- 只有认证后才能开通微信支付功能
- 只有认证后才能使用更多接口能力
- 认证后用户信任度更高

#### 第三步：设置小程序基本信息

- [ ] 小程序名称：建议使用 "爱上杜美人" 或包含业务关键词的名称
- [ ] 小程序头像：准备一张 JPG/PNG 图片（120x120像素）
- [ ] 小程序简介：简单描述业务内容
- [ ] 服务类目：选择 "商业服务 → 租赁服务" 或相关类目

#### 第四步：提供账号信息给我

认证完成后，请在小程序后台获取以下信息发给我：

- [ ] **AppID**（小程序ID）
  - 位置：设置 → 开发设置 → 开发者ID
  - 格式：wx开头的一串字符，如 wx1234567890abcdef

- [ ] **AppSecret**（小程序密钥）
  - 位置：设置 → 开发设置 → 开发者ID
  - 点击"生成"按钮获取（只显示一次，请妥善保存）

### 💰 费用说明

- **认证费：** 300元/年（微信官方收取）
- **无其他费用**

### ⏰ 预计完成时间

- 注册：30分钟
- 企业认证：1-3个工作日（含审核时间）

### ⚠️ 注意事项

1. 一个邮箱只能注册一个小程序
2. 小程序名称不能与现有小程序重复
3. AppSecret非常重要，请妥善保管，不要泄露给他人
4. 认证费每年都需要续费

## 6. 开通微信支付

### 📝 工作说明

开通微信支付后，用户才能在小程序中完成押金支付、购买等操作。

### ✅ 需要做什么

#### 第一步：申请微信支付商户号

- [ ] 访问微信支付商户平台（https://pay.weixin.qq.com/）
- [ ] 点击"注册微信支付商户号"
- [ ] 选择主体类型：**企业**
- [ ] 填写企业信息和联系人信息
- [ ] 上传营业执照、对公账户信息、法人身份证等材料
- [ ] 填写经营信息（经营类目、商品描述等）
- [ ] 提交审核

#### 第二步：签约并绑定小程序

- [ ] 审核通过后，登录微信支付商户平台
- [ ] 在"产品中心"中找到"APPID授权管理"
- [ ] 添加您的小程序AppID进行关联
- [ ] 签署《微信支付服务协议》

#### 第三步：提供商户信息给我

- [ ] **商户号（MchID）**
  - 位置：账户中心 → 商户信息
  - 格式：纯数字，如 1234567890

- [ ] **下载商户证书**
  - 位置：账户中心 → API安全 → 申请API证书
  - 下载后将证书文件发给我（我会妥善保管）

### 💰 费用说明

- **开通费用：** 免费
- **交易手续费：** 0.6%（即用户支付100元，您实际收到99.4元）
- **提现费用：** 免费（但有单笔限额5万元）

💡 **示例：** 如果一天收入1000元，手续费为6元

### ⏰ 预计完成时间

- 提交申请：30分钟
- 审核时间：1-5个工作日

### ⚠️ 注意事项

1. 经营类目要选择正确，建议选择"商业服务-租赁服务"
2. 对公账户信息必须准确，提现会打入该账户
3. 商户证书非常重要，请妥善保管
4. 如果审核不通过，微信会发邮件说明原因，补充材料后重新提交
